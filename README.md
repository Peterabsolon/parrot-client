# My awesome boilerplate

## Development

```bash
yarn dev
```

## Production

```bash
yarn build
yarn start
```

## Stack

- TypeScript
- Next.js
- MobX
- GraphQL
- styled-components
- rebass (UI primitives)

## Extra

- Jest
- @testing-library/react
- ESLint with Airbnb preset
- Prettier
- GraphQL codegen (for TS)
